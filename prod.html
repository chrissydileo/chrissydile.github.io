Formatting Pet Tags

Formatting Pet Tags:

Formatting Pet Tags for the laser includes using several procedures:


Tag Flipper will move important data to the non-embossed side of Cat and Paw shaped tags.
PrintMerge in Corel will place the data onto the appropriate lines of the corresponding tag shape.
AutoResizer in Corel will go through each tag of a printmerged file and resize it automatically based on shape and size.
Manual Resizer in Corel is a tool to help you go through tags after AutoResizer and make any needed changes.

Follow the procedures below to format Pet Tag products to be ready for the lasers:

  1. Tag Flipper

    Move important data to the non-embossed side of Cat and Paw shaped tags.

    1. On the P Drive, go to the 'For Production' Folder: P:\Pet Tags\For Production
    2. Locate the date folder for the data you need to format, and make a COPY of the excel file that says FINISHED
    3. Open the COPY of the FINISHED file and click the 'Tag Flipper' Button.
    4. "Follow the instructions in the Tag Flipper tool. You will be shown data for the front and back engraving for any PAW, SMP, CAT, and SMC tags. These tags are all embossed on the BACK side.
      5. *If the customer entered a majority of the contact info (phone, address, etc.) on the BACK, click FLIP.
      *If the tag is fine formatted as is, click NEXT.
      *If you accidently flipped, or didn't flip a tag and need to go back, click BACK."
    5. Once you have gone through each tag in each shape, a pop-up will appear asking if you're ready to Create CSV Files. Click YES. This will create 4 new CSV files for these embossed shapes that reflect any changes you made to the data. These files will appear in the same folder as the copied FINISHED files.


  2. PrintMerge

    1. Locate the Print Merge File for that product: Go to: P:\AMZ-PACKAGE\COREL\PET (Or click the link below) Then select the Corel file that starts with “PM-”, and ends with the product you are merging. (example: PM-HYD would be used to merge the HYD csv file)
    2. After opening the corresponding file, select the 'File' > 'Print Merge' > 'Create/Load Merge Fields...' (IMAGE A - Green Oval)
    3. In the Print Merge Wizard: Select Import text from a file or ODBC data source, then click NEXT.
    4. On the next screen navigate to the corresponding CSV file in the date folder’s Pet CSV folder. Be sure to change to file type (Circled in purple IMG B) to Comma Separated Files (.CSV) or the files will not show in the explorer. *NOTE: If you are formatting PAW, SMP, CAT, or SMC, be sure to use the flipped CSV file that you created in part 1.
    5. Click NEXT 2 more times, then FINISH.
    6. Go back to the Print Merge Menu (IMAGE A), and select 'Merge to New Document.' (Red Oval)

      7. *The new document is now ready for AutoResizer.


  3. AutoResizer

    1. Once a CSV file has been merged with it's Corel Template, click the 'openprovmenu' Button on your tool bar. (IMAGE C - Blue Circle)
    2. In the Product Formatting Menu that opens, select push the 'PET: RESIZE' Button.
    3. Select the shape you are formatting from the drop down menu, Then press 'GO'.
    4. Save the File in the corresponding date folder in For Production.

      5. *The document is now ready for Manual Resizing to check for needed edits before sending to the laser.
