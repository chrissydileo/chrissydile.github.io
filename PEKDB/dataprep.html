Run ProvAuto

This is the application used to download and prep data for Amazon and Etsy (Through AutoEtsy) orders ProvAuto should be run each morning Mon-Sat and again each day around Noon on Tue-Fri.

  1. Open ProvAuto, located in: P:\AMZ-PACKAGE\ProvAuto Or click below:


  2. Click START in ProvAuto and follow the instructions.

*NOTE: If this is your first time running ProvAuto from a new computer follow the steps below to install 7-zip before running: Go to 7-zip.org by clicking the Download 7zip Link for Windows 32-bit x86 Follow the installation instructions. During the process it should confirm the file path which should be: C:\Program Files (x86)\7-Zip

Update Remake Database

This is the application used to import previously processed orders and serve as a database for remake orders to be pulled from. This should be run at least once a week.

Open the Amazon Order database located in: P:\AMZ-PACKAGE\AmazinPS\AMZ ORDERS, then follow instructions in the program which are as follows:

  1. Press the 'Import Order Data' button. When the import is complete, the 'RECENTLY ADDED FILES' List should be updated with all the files currently in the Dbin folder.
  2. Press ctrl + s or select file -> save.
  3. Delete the data sets from the Dbin folder listed on the 'RECENTLY ADDED FILES' List.
  4. Close the AMZ ORDERS file.

Merging Office Packing Lists

  1. Open the Publisher Template file in : P:\AMZ-PACKAGE\Packing Lists\PL_Creator Office
  2. Click “YES” when asked if you would like to load the previous data source. (*should point to P:\AMZ-PACKAGE\Packing Lists\Sample 9Item PackingList)
  3. Click the "Mailings" Tab and select "Edit Recipient List" (Image 1)
  4. Navigate to the file you'd like to merge. (Will be in the working dates' folder, within the packing list folder)
  5. Select the first table. Make sure "1st row contains headers" is checked. Click OK.
  6. De-select the Sample customer, then Click OK.
  7. In the Mailings tab, click "FINISH AND MERGE", then "MERGE TO NEW PUBLICATION".
  8. After merging is complete, Save the file as a PDF. Go to File > Save As > Change file type to ".PDF" and save the PDF file in corresponding days Packing List Folder.

Print Pet Packing Guides

  1. Go to the Date Folder in For Production
  2. Open the Pet Packing Guide File inside the Packing List Folder.
  3. Select columns A through V, and press ctrl+p or or go to File > Print.
  4. For Printing Options Select the following:
    • Printer: Print to PDF -OR- Directly to your printer.
    • Print Selection
    • Landscape Orientation
    • Narrow Margins
    • Fit All Columns on One Page
  5. Press Print. *If you are printing to PDF save the PDF in the PackingList folder for that date.
